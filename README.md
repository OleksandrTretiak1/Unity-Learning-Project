# Unity Essentials Lab Project — Третяк Олександр

Цей репозиторій містить виконання навчальних модулів з курсу Unity Essentials. 

## Виконані модулі:

### 1. 3D & Audio Essentials
* **Опис:** Створено інтер'єр кімнати, налаштовано матеріали, освітлення, додані фонові звуки та музика.

### 2. Programming Essentials
* **Ключові механіки:**
    * **Керування гравцем:** Реалізовано переміщення та стрибок через `Input System Package`. Стрибок налаштовано за допомогою `AddForce` для фізичної коректності.
    * **Система збору предметів:** Пилосос збирає пилинки (що мають анімацію обертання) при зіткненні. Налаштовано так, що інші об'єкти не можуть з ними взаємодіяти.
    * **Взаємодія з оточенням (Двері):** Реалізовано механіку відкривання дверей, що спрацьовує при наближенні гравця.
    * **Цикл дня і ночі:** Створено скрипт `DayNightCycle`, який обертає `Directional Light`. Швидкість зміни часу доби можна редагувати безпосередньо в **Inspector**.
